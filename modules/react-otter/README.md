# react-otter

Exports an interface for using Otter with React.

## Installation

Add `@jsdotlua/react-otter` in your dependencies:

```bash
yarn add @jsdotlua/react-otter
```

Or if you are using `npm`:

```bash
npm install @jsdotlua/react-otter
```
